# Calculadora Simples em Python

Este é meu primeiro projeto em Python: uma calculadora simples feita para praticar conceitos básicos da linguagem.

## 📌 O que o programa faz

O programa pede ao usuário:

- Um operador matemático (`*`, `/`, `+`, `-`)
- Dois números

Em seguida, ele realiza a operação escolhida e exibe o resultado.

## 🧠 Conceitos praticados

Neste projeto, pratiquei:

- Entrada de dados com `input()`
- Conversão de tipos (`float`)
- Estruturas condicionais (`if`, `elif`, `else`)
- Operações matemáticas básicas
- Tratamento simples de erro (como divisão por zero)

## ⚙️ Como executar

1. Tenha o Python instalado (versão 3+)
2. Execute o arquivo `.py`
3. Siga as instruções no terminal

## Exemplo:

Escolha uma operação (* / + -): *
Digite o primeiro número: 5
Digite o segundo número: 3
O resultado é: 15
