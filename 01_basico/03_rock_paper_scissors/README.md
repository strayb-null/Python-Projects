# 🪨📄✂️ Pedra, Papel, Tesoura

Jogo clássico de Pedra, Papel e Tesoura contra o computador, construído com Python e Tkinter.

---

## 📸 Sobre o projeto

Projeto desenvolvido como parte do roadmap de estudos em Python. O objetivo foi praticar lógica de programação, estrutura de funções e construção de interfaces gráficas com Tkinter.

---

## ⚙️ Tecnologias

- Python 3
- Tkinter — interface gráfica
- Random — geração de escolha aleatória do computador

---

## 🎮 Como funciona

1. O usuário digita sua escolha: `pedra`, `papel` ou `tesoura`
2. O computador sorteia uma escolha aleatória
3. O resultado aparece na tela: vitória, derrota ou empate
4. É possível resetar o jogo ou sair

## 💡 Aprendizados

- Criação de janelas com `Tk()` e configuração de layout com `.place()` e `.pack()`
- Uso de `StringVar` para vincular variáveis à interface
- Lógica de comparação com `if/elif`
- Boas práticas: geração aleatória dentro da função para evitar resultado fixo
