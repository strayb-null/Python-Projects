# 📖 Mad Libs CLI - Gerador de Histórias Interativo

Este é um projeto de **Mad Libs** (história de palavras faltantes) feito em Python. O programa pede palavras ao usuário e as insere em uma história pré-definida, criando um resultado divertido e único a cada execução.

## 🎯 O que o programa faz

O programa solicita ao usuário:

- Um nome de amigo
- Um esporte
- Uma cidade
- Um jogador famoso
- Uma bebida
- Um lanche

Com base nessas entradas, ele gera uma história sobre um "dia memorável" praticando esportes.

## 🧠 Conceitos praticados

Neste projeto, pratiquei:

- Entrada de dados com `input()`
- Armazenamento de dados em variáveis
- Formatação de strings com **f-strings**
- Funções (`def`)
- Estrutura de repetição (`while`)
- Estruturas condicionais (`if`, `elif`, `else`)
- Uso do módulo `time` (para `time.sleep()` – efeito dramático)
- Menu interativo no terminal
- Bloco `if __name__ == "__main__"` (ponto de entrada)
- Criação de ASCII art para o menu
