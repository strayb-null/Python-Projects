# 🎮 Jogo do 21 (Terminal)

Um pequeno jogo em Python para jogar contra o computador no terminal.

## 📜 Regras

- Os jogadores falam números em sequência começando do 1.
- A cada turno, você pode dizer **1, 2 ou 3 números consecutivos**.
- Quem falar o número **21 perde**.

## 🤖 Computador

O computador usa uma estratégia baseada em múltiplos de 4 para tentar vencer.

💡 Dica: tente deixar o adversário nos números:
**4, 8, 12, 16, 20**
