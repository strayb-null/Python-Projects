# 🎯 Jogo de Adivinhação

O computador sorteia um número entre 1 e 100. Você tem tentativas ilimitadas para adivinhar — a cada chute, o jogo diz se o número secreto é maior ou menor.

---

## Conceitos praticados

- `while True` com `break` para loop até acertar
- `try/except ValueError` para tratar entrada inválida
- `if/elif` encadeado para comparação
- `random.randint()` para gerar número aleatório
- f-strings para formatar a mensagem final

---

## O que aprendi

- `input()` sempre retorna string — é preciso converter com `int()`
- `try/except` é a forma idiomática de validar conversões em Python
- Um `else` no final de `if/elif` pode ser código morto se os casos já cobrem tudo
